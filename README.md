# HeicToPng.org - Client-Side HEIC Converter

A high-performance, client-side HEIC to PNG converter built with Next.js 14+, Tailwind CSS, and heic2any.

## Features

- **Client-Side Only**: Uses `heic2any` to convert files directly in the browser. No server uploads.
- **Unlimited & Free**: No limits on file size or count.
- **Batch Processing**: Convert multiple files at once.
- **Privacy Focused**: Files never leave the user's device.
- **Responsive Design**: Mobile-first UI using Tailwind CSS.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Core Libraries**: `heic2any`, `jszip`, `file-saver`

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Build for Production

```bash
npm run build
npm start
```
